Getting Started

Compiling

Let's first write a simple hello world code. You may use any text editor for that, but we suggest using Visual Studio Code with the RISC-V Support extension.

1. Create a file named "hello.s" with the following content: 

.globl _start
  
    _start:
        li a0, 1            # file descriptor = 1 (stdout)
        la a1, string       # buffer
        li a2, 18           # size
        li a7, 64           # syscall write (64)
        ecall
    
    _end:
    
        li a0, 0            # exit code
        li a7, 93           # syscall exit
        ecall
    
    
    string:  .asciz "Hello RISC-V ALE!\n"

The easiest way of compiling code for RISC-V ALE is using clang, which can be installed on most Debian-based Linux distributions using the following command: 

    $ sudo apt install clang lld

2. Compile the code:

    $ clang --target=riscv32 -march=rv32gc -mabi=ilp32d -mno-relax hello.s -c -o hello.o
    $ ld.lld hello.o -o hello.x

Running

  1. Click on the upload button () at the top-right corner of this page.
  2. Select the file hello.x
  3. Click on the "RUN" button (also at the top-right corner of this page)

A terminal will open and you should see the output of your code's execution.

To exit this tutorial, refresh the page.
